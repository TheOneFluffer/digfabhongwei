Fusion 360 Design for Display Case (3D Printer):

Using a 3D printer to design the Caddy that will be found at the bottom part of the display case to allow the electronics to sit on top of it, ensuring the battery replacement process to be easier and make the wiring process a breeze

Specs for Caddy:

First, I decide on some dimensions that I want for my display case (A typical small figurine is around 250 to 280 mm in height) The space available in the caddy area is ~40mm in height

Length: 121 mm

Width: 125 mm

Height: 10 mm

Thickness: 2 mm

 

Figure 1: User params for Caddy

Choose the starting point on top of base and sketch the dimensions of the shape (121 x 125 mm)

Figure 2: Sketching caddy design

Add a rectangle near the front to act as a handle to pull the caddy out and use the fillet tool to round off the edges to make it rounder (Handle fillet: 2mm and surface fillet: 3mm)

Handle height: 4mm

Figure 3: Handle and rounded corners

In real life, I use my maker uno to get the exact locations of the screw holes so that my uno can sit in there and not rattle around.

Figure 4: Marking holes on the piece of paper

What I learned from the paper cutout:

The size of the hole is around 3mm in diameter (I choose ard 2.5mm for tolerance)

Figure 5: Recreating it to the best of my ability

Next, I select the pegs and extrude them to around 5mm in height

After creating the caddy in Fusion 360, You can export it as a .STL file (binary) You can find the option from Utilities > Make > 3D Print and select the parameters to send to cura

Figure 6: Exporting to Cura

Maker UNO Programming Logic:

Push button is to toggle modes: 0 - 4

When mode 0, do not turn on

If it is mode 1: turn red

If it is mode 2: turn green

If it is mode 3: turn blue

If it is mode 4: display rainbow color

Button only has 2 states, 1 and 0

I have to use a counter to count amount of time button is pressed

if button is pressed 5 times, it will reset back to 0, and the circuit will turn off

Also need to configure 15ms for button debouncing
